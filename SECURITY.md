# Security Policy

## Supported Versions

Atualmente, apenas a versão mais recente do projeto recebe correções e atualizações de segurança.

| Version | Supported |
| ------- | ---------- |
| Latest  | ✅ |
| Older versions | ❌ |

---

## Reporting a Vulnerability

Se você encontrar alguma vulnerabilidade, falha de segurança ou comportamento suspeito no projeto, pedimos que reporte de forma responsável.

### Como reportar

Abra uma issue privada (quando possível) ou entre em contato com os mantenedores do projeto através do GitHub:

- Arena Dev Team
- Repositório: [https://github.com/ProjetoArenaDev/Havan](https://github.com/ongocasite/ONG-OCA-SITE)

### O que incluir no relatório

Tente informar:

- Descrição da vulnerabilidade
- Passos para reproduzir
- Impacto esperado
- Screenshots ou logs (se necessário)

### Tempo de resposta

Tentaremos analisar os relatórios em até 7 dias.

### Política de divulgação

Pedimos que vulnerabilidades não sejam divulgadas publicamente antes da análise e correção pela equipe responsável.
